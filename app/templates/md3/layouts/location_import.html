description CSV-Format

Die CSV-Datei sollte folgende Spalten enthalten:

Spaltenname Beschreibung Pflichtfeld
Name Name des Standorts Ja
Straße Straße und Hausnummer Nein
PLZ Postleitzahl Nein
Stadt Stadt/Ort Nein
Bundesland Bundesland/Region Nein
Größe Größe in Quadratmeter (numerisch) Nein
Sitzplätze Anzahl der Sitzplätze (numerisch) Nein
Beschreibung Zusätzliche Informationen Nein
Bild URL oder Pfad zum Bild Nein
Latitude Breitengrad (Dezimalzahl) Nein
Longitude Längengrad (Dezimalzahl) Nein

code Beispiel CSV-Datei

Beispiel für eine korrekt formatierte CSV-Datei:

Name;Straße;PLZ;Stadt;Bundesland;Größe;Sitzplätze;Beschreibung;Latitude;Longitude
Hauptsitz Berlin;Unter den Linden 1;10117;Berlin;Berlin;500;100;Hauptsitz;52,518611;13,376111
Filiale Hamburg;Speicherstadt 7;20457;Hamburg;Hamburg;250;50;Filiale Nord;53,544389;9,993333
Filiale München;Münchnerstr. 15;80331;München;Bayern;180;30;Filiale Süd;48,137154;11,576124

lightbulb Tipps für den Import

  • Stellen Sie sicher, dass die erste Zeile die Spaltennamen enthält
  • Verwenden Sie das richtige Trennzeichen (meist Semikolon bei deutschen Excel-Dateien)
  • Latitude und Longitude sollten als Dezimalzahlen angegeben werden (z.B. 52.518611)
  • Leere Felder sind erlaubt, außer für das Pflichtfeld "Name"
  • Bei Fehlern wird eine detaillierte Fehlermeldung angezeigt
